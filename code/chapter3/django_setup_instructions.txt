
# Instructions to create the Django project and app

1. Install Django if not already installed:
   ```bash
   pip install django
   ```

2. Create a new Django project:
   ```bash
   django-admin startproject backend_basics
   cd backend_basics
   ```

3. Update the `views.py` file in the backend_basics app with the following:
   ```python
   from django.http import JsonResponse

   def home(request):
       return JsonResponse({"message": "Welcome to the Backend Basics API with Django!"})

   def recommendations(request):
       return JsonResponse({"recommendations": ["Movie 1", "Movie 2", "Movie 3"]})
   ```

4. Map URLs in the `urls.py` file:
   ```python
   from django.urls import path
   from .views import home, recommendations

   urlpatterns = [
       path('', home),
       path('recommendations/', recommendations),
   ]
   ```

5. Run the server:
   ```bash
   python manage.py runserver
   ```
